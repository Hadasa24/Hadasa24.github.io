NUESTROS PRODUCTOS

Somos una empresa que posee una gran trayectoria en el mercado.
Nos enfocamos en el servicio que reciben nuestros clientes y en la evolución de nuestros productos.

SYSCARD

Software completo para el manejo de toda la operativa que posee una tarjeta de crédito.

SYSDEBITO

Módulo que permite controlar las tarjetas de Débito generadas por las entidades bancarias.

SYSCARD WEB

Ofrece las mismas funcionalidades que el Syscard pero el aplicativo esta desarrollado para acceder por medio de un navegador Web.

SWITCH (Debito y/o Crédito)

Servicio WCF que permite el flujo, validación y control de autorizaciones entre múltiples canales de comunicación.

TRANSACSERVER

Programa Instalado como servicio en un servidor Windows; actúa como monitor de transacciones entre el cliente y el servidor.

CORE BATCH

Software que permite la automatización de la ejecución de los procesos batch multiplataforma, en base a parámetros de ejecución predefinidos por el administrador.
